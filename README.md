# Smart Environmental Monitoring System

An Arduino-based embedded monitoring system that uses an LDR sensor for real-time environmental light detection and multi-state LED indication.

## Features
- Real-time light intensity monitoring
- Multi-state LED indication
- Moving average filtering
- Threshold validation
- Fault detection handling
- Serial debugging logs

## Components Used
- Arduino Uno
- LDR Sensor
- LEDs
- Resistors
- Breadboard
- Jumper wires

## System Workflow

```
LDR Sensor
↓
Analog Signal Reading
↓
Moving Average Filtering
↓
Threshold Validation
↓
System State Detection
↓
LED Output + Serial Debugging
```
